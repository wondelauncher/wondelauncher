FAQ

Быстрые ответы по запуску и установке.

Почему игра может выйти с code 1?

Обычно это Java/моды/неполные файлы профиля. Smart Repair чинит runtime автоматически.

Где хранятся кастомные версии?

В папке AppData\Roaming\.wondelauncher\versions.

Можно ли использовать видео как фон?

Да. Поддерживаются MP4/WEBM, и видео повторяется циклически.

Есть оффлайн режим?

Да, а для онлайн-режима доступен вход Microsoft.
